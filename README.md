# Sentence-classification-system

A project for classifying sentences and text into one of four classes using Naive Bayes Theorem

# 📌 Features

1.Classification of test data using Naive Bayes

2.Can be used to mass detect class of text data

3.User friendly interface

4.Fast and accurate execution

# 📌 Objective

The primary objective is to train the algorithm to accurately assign new, unseen text into one of four distinct target labels:

•	Technology

•	Sports

•	Politics

•	Entertainment

# 📌 How it works

a.Input:

User enters a sentence

b.Process:

The text data gets converted into a matrix of token counts,then the  model predicts the labels for the test data

c.Output:

The class of the sentence is displayed
